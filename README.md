## Usage
In this apllication you can:
* View Employees
* View Employees by Department
* Add Employee
* Remove Employees
* Update Employee Role
* Add Role

## Built With
* inquirer
* mySQL
* console.table
* Javascript
* Node.js

## Installation
1. npm init -y 
2. npm i
3. npm i inquirer
4. npm i mysql2
5. npm i console.table
6. run .sql file in mySQL workbench 
7. run node server.js
